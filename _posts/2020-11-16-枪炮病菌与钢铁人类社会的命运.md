# 枪炮\病菌与钢铁:人类社会的命运



本书中的主题就是**为什么在不同大陆上人类以如此不同的速度发展?**

不同民族之间相互作用的历史,就是通过征服\流行病和灭绝种族的大屠杀来形成现代世界的.

"不同民族的历史遵循不同的道路前进,其原因是民族环境的差异,而不是民族自身在生物学的差异"

粮食生产是枪炮\病菌和钢铁发送的一个先决条件.

定居部落由于没有在迁移途中携带小孩子的这种问题的限制,他们可以多生多养.

总之,动植物的驯化意味这人类的粮食越多,因而也意味这人口越来越稠密,因此带来的粮食剩余和利用蓄力运输剩余的粮食,成了定居\行政上集中统一的,社会等级分明\经济上富有个性精神的社会发展的先决条件.

每一种候补的野生动物都必须具有许多不同的特点,缺少哪一个必不可少的特点,都会使驯化失败.

病菌传播的最不费力的方式,就是等待着被动的传染给下一个受害者.

为什么农业的出现会成为我们群众传染病形成的开端?

1. 农业比狩猎采集的生活方式维持高得多的人口
2. 农业人口定居,排泄物有大量的病菌,而狩猎采集经常变换营地.



3个基本策略构成书写系统的基础:

- 字母为语言每一个基本的音提供一个独一无二的符号.
- 利用所谓的语标,用一个书写符号来表示一个完整的词
- 一个符号代表一个音节.



为什么文字在某些社会出现并向某些社会传播,但不向其他许多社会传播?

- 早起书写系统的有限容量
- 有限用途
- 有限使用者



主要结论:

- 技术的发展是长期积累,而不是靠独立的英雄行为
- 技术在发明出来后大部分都得到了使用,而不是发明出来去满足某种预见中的需要



社会至少有四个因素影响对发明的接受:

1. 与现有技术相比较的相对经济利益
2. 社会价值和声望
3. 是否符合既得利益
4. 新技术的优点能够很容易地看到.



其中5个因素涉及社会的经济和组织:

- 古典时期可以得到的廉价的奴隶劳动,妨碍了当时的发明创造.
- 保护发明者的所有权的专利和其他财产奖励发明
- 现代工业社会提供大量的技术培训机会.
- 美国社会强烈的个人主义允许发明者为自己赚钱.



另4个解释是意识形态方面:

1. 为创新必不可少的冒险
2. 科学观点
3. 对各种观点和异端观点的宽容.
4. 宗教在其与技术创新的关系上差异很大.



一个上层人物仍然保持比平民舒服的生活方式时,要怎样做才能获取群众的支持呢?

1. 解除平民的武装,同时武装上层掌权的人物
2. 用通行的方法把得到财务很大一部分再分配个群众来博取他们的欢心.
3. 利用对武力的绝对控制来维持公共秩序和制止暴力以促进社会幸福.
4. 盗jue统治者.制造一种为盗jue统治辩护的意识形态或宗教.
